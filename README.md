## Project Overview

This project analyzes customer churn behavior 
using real transaction data from Olist, a 
Brazilian e-commerce marketplace, covering 
99,441 orders from September 2016 to 
October 2018.

The goal is to identify why customers don't 
return, segment them by value, and provide 
actionable business recommendations.

---

## Key Findings

| Metric | Value |
|--------|-------|
| One-Time Buyer Rate | 97% |
| Overall Churn Rate | 59.2% |
| Month 1 Cohort Retention | 0.47% |
| Repeat Customer Avg Spend | R$ 308 vs R$ 160 |
| Champions vs Lost CLV Gap | 8.6x |
| Revenue at Risk | R$ 5,821,138 |
| CLV Uplift Potential | R$ 2,077,785 |
